# Navigation

## Purpose

The navigation system helps users move naturally through the Living Knowledge Network while minimizing cognitive load. It is designed around how people discover, contribute to, and evolve knowledge.

## Navigation Principles

- Keep primary navigation simple and consistent.
- Prioritize conversations over documents.
- Make AI assistance accessible from anywhere.
- Surface relevant knowledge through context.

## Public Navigation

- Home
- Discussions
- Experts
- Insights
- Pricing

## Authenticated Navigation

- Discover
- Domains
- Conversations
- Knowledge Network
- AI Assistant
- My Workspace

## Design Goals

- Fast discovery
- Clear hierarchy
- Minimal clicks
- Scalable information architecture
- Seamless transition between browsing and contributing
